# XOR Probleminin Yapay Sinir Ağları ile Çözümü
Bu proje, Python ve NumPy kullanarak 2-2-1 mimarisine sahip bir yapay sinir ağı ile XOR probleminin çözümünü içermektedir.

# Mimari Detayları:
- Giriş Katmanı: 2 nöron
- Gizli Katman: 2 nöron (Sigmoid Aktivasyonu)
- Çıkış Katmanı: 1 nöron (Sigmoid Aktivasyonu)
